05 / Apprendre

Ma manière d'apprendre.

À 42, on apprend par la pratique et en allant chercher les réponses. L'IA est un outil puissant et je l'utilise régulièrement ; pendant la phase d'apprentissage, j'essaie simplement de faire attention à la façon dont je m'en sers.

01

L'IA, surtout pour comprendre.

Je m'en sers pour reformuler un concept, comparer des approches ou décoder un message d'erreur. Sur les sujets que je suis en train d'apprendre, je préfère écrire le code moi-même, pour que l'apprentissage reste actif.

02

Croiser avec la doc.

RFC, man pages, documentation officielle : j'y retourne régulièrement. L'IA répond vite, la doc répond juste, et je m'appuie sur les deux selon ce que je cherche.

03

Comprendre ce que je commit.

J'essaie de pouvoir expliquer chaque ligne de mon code. Quand j'hésite sur une suggestion, je relis la doc ou je prends le temps de comprendre comment elle fonctionne avant de l'intégrer.

Mes références au quotidien : man RFC 7230 cppreference POSIX.1-2017 LKD & TLPI Stack Overflow
